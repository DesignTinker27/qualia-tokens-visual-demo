Usage Guidelines

When to Use Each Variant

  • Primary: Main call-to-action, limit to one per view
  • Secondary: Supporting actions, cancel buttons
  • Destructive: Delete actions, permanent changes
  • Success: Confirm actions, save changes
  • AI: AI-powered features, smart suggestions

React Code Example

import Button from './Button';

// Primary action
<Button variant="primary" size="medium">
  Save Changes
</Button>

// Secondary action  
<Button variant="secondary">
  Cancel
</Button>

// Destructive action
<Button variant="destructive">
  Delete Account
</Button>
